> ℹ️ 2026 제9회 지선에서 유권자 1명이 받는 투표용지는 보통 7종(②는 교육감, ③·④는 광역의원 비례/지역구, ⑤·⑥·⑦은 기초단체). 제주는 교육의원까지 8종.

## sgId (선거 식별자) 포맷
```
{sgTypecode 2자리 zero-pad}{yyyy}{MM}{dd}
ex) 0020260603  = 제9회 전국동시지방선거 (2026.6.3)
    0020250603  = 제21대 대통령선거    (2025.6.3)
    0020220601  = 제8회 전국동시지방선거 (2022.6.1)
    0020220309  = 제20대 대통령선거    (2022.3.9)
    0000000000  = 역대 통합 (info.nec.go.kr 한정)
```
> 지방선거처럼 하위에 sgTypecode별 세부 선거가 7~8개 붙는 경우, `subSgId`로 분기됩니다. 스키마에서는 `elections` ↔ `sub_elections` 1:N.

## 행정구역 코드 주의사항
- **세종시 town 코드 4148/4149 → 4128 치환** (info.nec.go.kr JS 내장 로직)
- **제9회 지선 cityCode 4600 → 2900 치환** (선거구 재획정 반영)
- 즉, 같은 "세종/대전" 지역이라도 연도별로 코드가 달라질 수 있음. `regions` 테이블에 `valid_from/valid_to` 필수.
